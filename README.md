# 🏋️‍♂️ Análisis y Limpieza de Datos de Entrenamiento (Pandas)

Este proyecto forma parte de mi aprendizaje en el **BOOTCAMP 2026**. En este ejercicio, he trabajado con un conjunto de datos real (un archivo CSV con registros de entrenamiento) que presentaba diversos errores típicos del mundo real.

## 🛠️ Lo que he implementado:
* **Limpieza de Datos:** Identificación y eliminación de filas con valores nulos (`NaN`).
* **Imputación de Valores:** Uso de la **media** para rellenar datos faltantes en la columna de Calorías mediante diccionarios y `.fillna()`.
* **Corrección de Formatos:** Conversión de fechas irregulares al formato estándar `datetime`.
* **Lógica de Negocio:** Filtrado de sesiones con duraciones incoherentes (errores de registro).
* **Análisis Estadístico:** Localización de récords personales usando `.idxmax()`.

## 🚀 Tecnologías utilizadas:
* **Python 3**
* **Pandas** (Librería principal para análisis de datos)
* **Git/GitHub** (Control de versiones y despliegue)

## 📊 Conclusiones clave:
